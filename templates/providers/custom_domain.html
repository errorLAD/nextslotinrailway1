Step-by-Step Cloudflare Setup
  1. Go to Cloudflare Dashboard → Select your domain → Click DNS in the sidebar
  2. Add CNAME Record:
    • Click "Add record"
    • Type: Select CNAME
    • Name: Enter {{ provider.custom_domain|get_subdomain }}
    • Target: Enter {{ cname_target }} Your unique target
    • Proxy status: ☁️ Proxied (Orange cloud ON)
    • Click Save
  3. Add TXT Record:
    • Click "Add record"
    • Type: Select TXT
    • Name: Enter {{ txt_record_name }} Your unique TXT name
    • Content: Paste {{ provider.domain_verification_code }}
    • Click Save
  4. Configure SSL: Go to SSL/TLS → Set encryption mode to "Full (Strict)"
SSL Configuration

In Cloudflare SSL/TLS settings, set encryption mode to "Full (Strict)" for maximum security.
